# PDF Report Generator (NodeJS)

[Example of generated PDF](https://drive.google.com/file/d/1HpUP7YzteUlv0lOVorjed3aelX8a4FQB)

---

### Technologies used

- **Backend and REST API**
  - NodeJS
  - Express
  - EJS
  - Nodemailer
  - Multer
- **Database**
  - MongoDB
- **Image Optimization**
  - Sharp
- **PDF Generation**
  - Puppeteer
- **Auth**
  - JSON Web Token
- **Server**
  - NGINX
  - PM2
- **Frontend**
  - JavaScript, CSS, HTML
